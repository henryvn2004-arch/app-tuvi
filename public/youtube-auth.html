🎬 YouTube Authorization

Authorize 1 lần để lấy Refresh Token.
Token này dùng mãi để auto-upload YouTube.

👆 Click để copy
Sau khi có token:
1. Copy refresh token ở trên
2. Vào Supabase → Settings → Edge Functions → Secrets
3. Thêm: YOUTUBE_REFRESH_TOKEN = [token]
4. Xong — YouTube upload sẽ dùng token này tự động
